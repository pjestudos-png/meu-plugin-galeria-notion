# 📚 Galeria Pro para RemNote

Transforme suas tabelas avançadas em uma experiência visual poderosa. Inspirado no layout de galeria do Notion, Para você que sente falta de uma galeria visual para suas coleções.

## ✨ Funcionalidades
- **Visual de Galeria:** Veja seus cartões como capas de livros, filmes ou animes.
- **Barra de Progresso Dinâmica:** Acompanhe seu avanço em tempo real (Heatmap).
- **Tradutor de Datas:** Datas automáticas no formato `📆 DD/MM/AAAA`.
- **Upload de Capas:** Suporte para imagens estáticas (JPG/PNG) e GIFs animados.
- **Filtros Avançados:** Organize por Status, Nota ou Gênero com lógica de exclusão (≠).

## 🚀 Como Instalar (Tutorial)
1. Vá em **Settings** -> **Plugins** no seu RemNote.
2. Clique em **Install from GitHub**.
3. Cole este link: `https://github.com/pjestudos-png/meu-plugin-galeria-notion`
4. Clique em **Install**.

## 📖 Como usar a Galeria
1. Crie um novo Documento (Pasta Pai)
2. Crie uma **Tabela Avançada** no RemNote.
3. Adicione uma **Tag** aos seus Rems (ex: #Estudos).
No título do Rem adicione a tag. (Ex. ##Estudos).
4. No plugin, clique em **+ Nova**, dê um nome e coloque a Tag exata (Ex. Estudos).
Tudo o que estiver na pasta pai, inclusive na tabela avançada, com a tag Estudos aparecerá dentro da biblioteca.
Volte a Pasta pai e copie o link (CRTL + R) e cole na URL e salve.
4. Para a Barra de Progresso, selecione as propriedades de "Progresso Atual" e "Total" nas configurações do plugin (ícone ⚙️).
para a barrinha de porcentagem funcionar a tabela tem que ter duas propriedades do tipo “Number” e ser colocada como propriedade nas opções total e progresso atual.
