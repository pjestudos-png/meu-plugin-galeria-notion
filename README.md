# 📚 Galeria Pro para RemNote

Transforme suas tabelas avançadas em uma experiência visual poderosa. Inspirado no layout de galeria do Notion, este plugin foi feito para quem sente falta de uma visualização organizada e estética para suas coleções.

## ✨ Funcionalidades
- **Visual de Galeria:** Veja seus cartões como capas de livros, filmes ou animes.
- **Barra de Progresso Dinâmica:** Acompanhe seu avanço em tempo real (Heatmap).
- **Tradutor de Datas:** Datas automáticas formatadas para `📆 DD/MM/AAAA`.
- **Upload de Capas:** Suporte para imagens (JPG/PNG) e GIFs animados.
- **Filtros Avançados:** Organize por Status, Nota ou Gênero com lógica de exclusão (≠).

## 📖 Como usar a Galeria
1. Crie um novo Documento (**Pasta Pai**).
2. Crie uma **Tabela Avançada** dentro deste documento.
3. Adicione uma **Tag** aos seus itens (ex: #Estudos). Dica: A tag deve estar no título do Rem.
4. No plugin, clique em **+ Nova**, dê um nome e digite a Tag exata (Ex: Estudos).
5. Copie o link da sua Pasta Pai (atalho `Ctrl + R` ou pelo menu de três pontos) e cole no campo **URL** das configurações da galeria.
6. **Barra de Progresso:** Para a porcentagem funcionar, sua tabela precisa de duas propriedades do tipo "Number". Selecione-as como "Progresso Atual" e "Total" no ícone da engrenagem (⚙️) do plugin.
